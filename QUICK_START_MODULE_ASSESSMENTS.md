# 🚀 Quick Start: Create a Course with Module Assessments

## The New 5-Step Process

### What You Need to Know

✅ You can create an ENTIRE course in one form  
✅ No more back-and-forth between pages  
✅ Assessments for EACH module (students must pass)  
✅ Final assessment only after ALL modules  
✅ All data saves when you submit

---

## Step-by-Step Guide

### STEP 1️⃣ : Basic Course Info (2 minutes)

Fill in your course details:

```
Title:           [Digital Marketing for Beginners]
Description:     [Learn the fundamentals of digital marketing from scratch...]
Category:        [Marketing ▼]
Level:           [Beginner ▼]
Duration:        [8 WEEKS]
Price:           [49.99]  (or 0 for free)
Banner Image:    [Upload thumbnail]

Click: "Next: Add Content" →
```

✅ **Pro Tip**: Make your title and description clear - students see this when browsing

---

### STEP 2️⃣ : Add Modules & Lessons (10-20 minutes)

Add multiple modules, each with multiple lessons.

**Example Course Structure:**

```
Module 1: Foundations (3 lessons, 45 min)
├─ Lesson 1: What is Digital Marketing? (10 min)
├─ Lesson 2: Digital Marketing Channels (20 min)
└─ Lesson 3: Setting Up Your First Campaign (15 min)

Module 2: Social Media Strategy (4 lessons, 60 min)
├─ Lesson 1: Facebook & Instagram Basics
├─ Lesson 2: Content Strategy & Planning
├─ Lesson 3: Analytics & Measurement
└─ Lesson 4: Real-World Case Studies

Module 3: Advanced Tactics (2 lessons, 30 min)
├─ Lesson 1: Paid Advertising Deep Dive
└─ Lesson 2: SEO Fundamentals
```

**For Each Module:**

```
Module Title:        [Module 1: Foundations]
Module Description:  [In this module you'll learn...]

Add Lessons (one at a time):
  ├─ Title:         [Lesson 1: What is Digital Marketing?]
  ├─ Video:         [https://youtube.com/watch?v=...]
  ├─ Duration:      [10 mins]
  ├─ Content:       [Overview of what we'll cover...]
  └─ Click: "Add Lesson to Module"

  [Repeat for Lesson 2, 3, etc.]

Click: "Save Module and Continue"

[Repeat for Module 2, 3, etc.]

Click: "Add Module Assessments" →
```

✅ **Pro Tip**: Keep lessons short (10-20 minutes). Students engage better with bite-sized content.

---

### STEP 3️⃣ : Create Module Assessments ⭐ **THIS IS NEW!** (15-30 minutes)

Now create an assessment for EACH module. Students must pass each one before continuing.

**Process:**

```
┌─ Select Module 1 (click on it)
│  └─ Add Assessment Details:
│     ├─ Title:           [Module 1 Assessment]
│     ├─ Description:     [Test your knowledge on digital marketing basics]
│     ├─ Passing Score:   [70]%
│     │
│     └─ Add Questions:
│
│        Question 1: Multiple Choice (10 points)
│        ├─ Text:    [What are the main digital marketing channels?]
│        ├─ Options:
│        │  ├─ Social Media
│        │  ├─ Email Marketing  ← CORRECT
│        │  ├─ Content Marketing
│        │  └─ All of the above
│        ├─ Points:     [10]
│        └─ Explanation: [All three are core digital channels...]
│
│        Question 2: True/False (10 points)
│        ├─ Text:  [SEO is only about keywords]
│        ├─ Answer: [False]  ← CORRECT
│        └─ Points: [10]
│
│        Question 3: Multiple Choice (10 points)
│        ├─ Text:  [What is the average CTR for display ads?]
│        ├─ Options: [...etc...]
│        └─ Points: [10]
│
│        Total: 30 points needed for passing
│        (70% = 21 points minimum)
│
│
├─ Select Module 2 (click on it)
│  └─ Add Assessment (same process)
│     ├─ 4 questions
│     └─ Passing Score: 70%
│
└─ Select Module 3 (click on it)
   └─ Add Assessment (same process)
      ├─ 3 questions
      └─ Passing Score: 70%

Click: "Continue to Final Assessment" →
```

✅ **Pro Tip**: Make assessments challenging but fair. Students should score 70-90% if they paid attention.

---

### STEP 4️⃣ : Final Assessment (10 minutes)

The final exam taken ONLY after students complete all modules.

```
Assessment Title:     [Final Comprehensive Assessment]
Description:          [Demonstrate mastery of all digital marketing concepts]
Passing Score:        [70]%  (Must be 70%+ for certificate)

Add Questions (5-10 questions recommended):

Question 1: Multiple Choice (20 points)
├─ Text: [Which channel drives the most ROI for B2B?]
├─ Options: [Facebook], [LinkedIn] ✓, [TikTok], [Pinterest]
└─ Points: [20]

Question 2: Multiple Choice (20 points)
├─ Text: [What's the first step in creating a digital strategy?]
├─ Options: [...etc...]
└─ Points: [20]

... (3 more questions to reach ~100 points total)

Total: ~100 points
70% passing = 70 points needed for certificate

Click: "Review Course" →
```

✅ **Pro Tip**: Final assessment should be comprehensive, covering all 3 modules. Increase difficulty.

---

### STEP 5️⃣ : Review & Publish (2 minutes)

Check everything looks good:

```
✓ Course Title: Digital Marketing for Beginners
✓ Modules: 3 (Foundations, Social Media, Advanced)
✓ Total Lessons: 9
✓ Module Assessments: 3 (with questions)
✓ Final Assessment: 5 questions

✓ Banner: [Thumbnail shows]

Click: "Publish Course" or "Submit for Approval"
   ↓
COURSE CREATED! 🎉
```

## FAQ

**Q: Can I edit the course after publishing?**
A: Yes, but changes might require re-approval. Contact support if needed.

**Q: What if a student fails 3 times?**
A: The course restarts. They begin from Module 1 Lesson 1.

**Q: Can students skip a module?**
A: No. They must complete modules in order.

**Q: Unlimited tries on final assessment?**
A: Yes! They can retake it as many times as needed to get 70%+.

**Q: How do certificates work?**
A: Automatically generated and emailed when they score 70%+ on final assessment.

**Q: Can I see student progress?**
A: Yes, check your instructor dashboard for analytics.
