# Recent Activities - Quick Reference Guide

## 🚀 Quick Start

### Where to Find Recent Activities?

**Option 1: Admin Dashboard**

```
Home → /admin → Scroll down → "Recent Activities" section
```

**Option 2: System Settings** (New!)

```
Sidebar → System Settings → "All System Activities" section
```

**Option 3: User Menu**

```
Profile icon (top-right) → Settings → System Settings page
```

## 🔍 How to Filter

**Click the filter button:**

```
[All Activities] [Registrations] [Approved] [Rejected] [Courses]
```

**What each shows:**

- **All Activities**: Everything
- **Registrations**: New users joining
- **Approved**: Instructors & courses approved
- **Rejected**: Rejected applications
- **Courses**: Course-related actions

## 🎯 Common Tasks

### Find who approved an instructor

1. Click **[Approved]** filter
2. Look for "instructor" in message
3. Check "Performed by" field

### See recent sign-ups

1. Click **[Registrations]** filter
2. View list (newest first)
3. Check timestamps

### Check rejection reasons

1. Click **[Rejected]** filter
2. Read rejection reason in red box
3. See who rejected it

### Track admin logins

1. Keep **[All Activities]** filter
2. Look for "logged in" message
3. Check time and admin name

### Monitor course actions

1. Click **[Courses]** filter
2. Review approvals/rejections
3. See feedback provided

## 🆘 Quick Troubleshooting

| Problem                     | Solution                                |
| --------------------------- | --------------------------------------- |
| No activities showing       | Click "All Activities" to reset filters |
| Pagination buttons disabled | Refresh page or less than 10 activities |
| Activities not updating     | Wait 1-2 seconds or refresh page        |
| Can't see specific activity | Check if wrong filter is active         |
| Timestamp seems wrong       | Check system clock on your device       |
